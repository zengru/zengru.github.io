槲栎

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：槲栎学名：Q. aliena简介：落叶乔木；树皮暗灰色，深裂。叶倒卵状椭圆形或长圆形，长10~20cm，有深波状齿，背面呈灰绿色。壳斗浅杯状，鳞片线状披针形。坚果长椭圆形或卵状球形。种子含淀粉，可酿酒，也可制凉皮等。

栓皮栎

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：栓皮栎学名：Q. variabilis简介：落叶乔木。叶互生；宽披针形，边缘具刺芒状锯齿，叶背灰白，密生细毛。壳斗碗状，鳞片反卷，包坚果2/3以上；坚果球形。树皮深灰色，木栓层厚，可用于软木塞、绝热、隔音和绝缘材料。

有柄石韦

发表于 2020-06-12 | 分类于 蕨类植物门 |
名称：有柄石韦学名：Pyrrosia petiolosa简介：多年生。根状茎细长，密被棕褐色鳞片。叶远生，厚革质；营养叶与叶柄近等长，卵圆形至长卵形，全缘；孢子叶短于叶柄，卵状椭圆形，下面密生深褐色的孢子囊群。生于岩石上或岩缝内。

枫杨

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：枫杨学名：Pterocarya stenoptera简介：落叶乔木。偶数（稀奇数）羽状复叶，小叶10~16，边缘有细锯齿。雌雄花均形成柔荑花序。果序下垂，小坚果近球形，具双翅，果翅长圆形至长圆状披针形。我国南部各地广布，常见栽培作行道树。

金钱松

发表于 2020-06-12 | 分类于 裸子植物门 |
名称：金钱松学名：Pseudolarix amabilis简介：落叶乔木。叶片条形，扁平，在长枝上螺旋状散生，在短枝上15~30枚簇生，向四周辐射平展，秋后变金黄色，圆如铜钱。球果10月成熟，种鳞熟时脱落。

马齿苋

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：马齿苋学名：Portulaca oleracea简介：多年生肉质草本，无毛，伏地铺散。叶互生或近对生，倒卵形。花小，黄色；萼片2；花瓣5，稀4；雄蕊通常8；柱头4~6裂。蒴果卵球形，盖裂。常见路边荒地，可作野菜食用。
