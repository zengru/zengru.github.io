鸢尾

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：鸢尾学名：Iris tectorum简介：多年生草本。叶剑形，锐尖，长3060cm，二行排列。花葶每分枝有花23朵，苞片倒卵状椭圆形；花蓝色，直径约10cm；花被6片，倒卵形，外被片有带斑纹的鸡冠状突起；雄蕊3；柱头裂片花瓣状。蒴果长椭圆形。各地庭院栽培。

山麦冬

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：山麦冬学名：Liriope spicata简介：多年生草本。具肉质小块根。叶禾叶状，宽46mm，中脉明显。花葶长于叶或少数短于叶（a）；花35朵生于苞腋内；花冠中部以上或顶端具关节；花被片矩圆形，淡紫色；花丝长约2mm；子房上位（b）。常见庭院栽培。

鹤望兰

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：鹤望兰学名：Strelitzia reginae简介：多年生草本。内质根粗壮；茎不明显。叶大，叶片矩圆形；有长柄。花茎顶生或生于叶腋。花形奇特，佛焰苞状，总苞紫色，花萼橙黄色，花瓣亮蓝色。为名贵的观赏植物，温室栽培。

韭莲

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：韭莲学名：Z. grandiflora简介：多年生草本。鳞茎卵形。叶条形，稍肉质，宽7~10mm。花茎稍短于叶；花单生，淡紫红色；花被管明显，裂片6；柱头3深裂。蒴果近球形。原产墨西哥南部和危地马拉，我国各地庭院栽培。

薯蓣

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：薯蓣学名：Dioscorea opposita简介：草质缠绕藤本。块茎圆柱形，长达1m。单叶互生至对生，三角状卵形或耳状3浅裂至深裂。花小，单性；雄花具6雄蕊。蒴果果翅半月形，宽约1.5cm，顶端及基部近圆形。校内野生及栽培，块茎即“山药”，供食用或药用。

麦冬

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：麦冬学名：Ophiopogon japonicus简介：多年生草本。小块根长11.5cm。叶禾叶状，宽1.53.5mm。花葶短于叶；花1~2朵生于苞片腋内；花梗近中部具关节；花被片6，披针形，白色或淡蓝紫色；雄蕊6，花丝很短；子房半下位。常栽培用于地面绿化。

蝴蝶花

蝴蝶兰

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：蝴蝶兰学名：Phalaenopsis aphrodita简介：附生。茎极短，扁平。叶椭圆状披针形至长椭圆状镰形，长1020cm。花葶弓曲，长2050cm或更长；花序总状；花大，形似蝴蝶，颜色华丽；唇瓣3裂，中裂片镘形，有2卷须。分布于我国台湾及爪哇一带岛屿，各地盆栽观赏。

垂笑君子兰

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：垂笑君子兰学名：C. nobilis简介：多年生草本。叶带状，宽2.55cm，质硬而粗糙，先端钝。伞形花序具2060朵花；花冠淡黄色至暗橘黄色，常低垂；花被片6，先端常呈绿色。是与君子兰同属的另一著名观赏植物。
