石楠

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：石楠学名：Photinia serrulata简介：常绿灌木或小乔木。叶革质，长椭圆形或倒卵状椭圆形，长922cm，疏生具腺细锯齿，先端尾尖。复伞房花序顶生；花瓣白色，近圆形。梨果球形，直径56mm，红色。为常见的园林栽培树种。

小天蓝绣球

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：小天蓝绣球学名：Phlox drummondii简介：一年生草本。叶卵形、矩圆形或披针形，全缘。聚伞花序顶生；苞片和小苞片条形；花冠高脚碟形，有红、白、黄、紫等色。各地庭院栽培。

小叶冷水花

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：小叶冷水花学名：Pilea microphylla简介：纤细小草本。茎肉质，多分枝。叶小，倒卵形至匙形，长3~7mm，先端钝。雌雄同株。雄花花被片4，雄蕊4；雌花较小，花被片3。生于溪边、石缝等潮湿环境中。

延羽卵果蕨

发表于 2020-06-12 | 分类于 蕨类植物门 |
名称：延羽卵果蕨学名：Phegopteris decursivepinnata简介：根状茎短而直立，被卵形鳞片。叶丛生，两面沿叶脉有疏针状毛和分枝毛。叶片倒披针形，顶部渐尖并 一至二回羽裂，羽片互生，基部以耳状翅相连，单一侧脉顶部着生圆形孢子囊群。生于溪沟两岸或路边。

火炬松

发表于 2020-06-12 | 分类于 裸子植物门 |
名称：火炬松学名：P. taeda简介：常绿乔木。树皮橘褐色，下部枝条开展下垂。针叶3针一束，长15~25cm，树脂道2，中生；叶鞘宿存。球果腋生，无柄，对称，卵状圆锥形；种子菱形，红褐色。原产美国东南部，我国江南各地引种栽培。

黑松

发表于 2020-06-12 | 分类于 裸子植物门 |
名称：黑松学名：P. thunbergii简介：常绿乔木。树皮黑灰色，新芽白色。针叶2针一束，长615cm，树脂道3，中生；叶鞘长约1.2cm。球果圆锥状卵形或卵圆形，有短柄，熟时栗褐色；鳞脐微凹或有短刺。种子倒卵状椭圆形，长57mm，种翅灰褐色。原产朝鲜和日本，国内引种栽培。

黄山松

发表于 2020-06-12 | 分类于 裸子植物门 |
名称：黄山松学名：P. taiwanensis简介：常绿乔木，树皮灰褐色。针叶2针一束，稍粗硬，长5~8cm。球果宽卵形，黄褐色；鳞脐背生，有短刺。著名林学家郑万钧认为黄山松与台湾松是同一物种，并采用后者的学名。分布于海拔800m以上山区，校园内有栽培。

牡丹

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：牡丹学名：Paeonia suffruticosa简介：落叶小灌木，株高多在0.52m。叶为二回三出复叶，顶生小叶常3深裂。花单生枝顶，直径1520cm；萼片5；花瓣数个至多数，倒卵形；雄蕊多数；心皮5，嫩时为革质的花盘包围。蓇葖果卵圆形。为著名观赏植物，品种花色极多。

牵牛

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：牵牛学名：Pharbitis nil简介：一年生缠绕草本，被粗硬毛。叶近卵状心形，常3裂。花冠漏斗状，白色、蓝紫色或紫红色。蒴果球形。原产热带美洲，各地栽培。

蜈蚣草

发表于 2020-06-12 | 分类于 蕨类植物门 |
名称：蜈蚣草学名：P. vittata简介：根状茎短，直立，被黄褐色鳞片；一回羽状复叶，矩圆形至披针形；羽片无柄，线状披针形，先端渐尖，基部截形或心形，时稍呈耳状。孢子囊群线形，沿羽片边缘着生。生钙质土或石灰岩上。
