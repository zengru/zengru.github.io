窄叶小苦荬

发表于 2020-06-26 | 分类于 被子植物门（双子叶合瓣花类） |
名称：窄叶小苦荬学名：I. gramineum简介：多年生草本，具乳汁，高6~30cm。茎自基部分枝。基生叶条状披针形，稀羽状分裂；茎生叶椭圆状披针形或披针形，渐尖，基部耳状抱茎。舌状花黄色。瘦果纺锤形，具喙；冠毛白色。生于田野、沟边和路边草地。

稻槎菜

发表于 2020-06-26 | 分类于 被子植物门（双子叶合瓣花类） |
名称：稻槎菜学名：Lapsana apogonoides简介：一年生草本，具乳汁，高1020cm。基生羽状分裂，长410cm，顶裂片较大，卵形，侧裂片3~4对。头状花序小；总苞椭圆形，长约5cm；花全部舌状，两性，花冠黄色。瘦果椭圆状披针形，扁；无冠毛。生于田间或路边。

红凤菜

发表于 2020-06-26 | 分类于 被子植物门（双子叶合瓣花类） |
名称：红凤菜学名：Gynura bicolor简介：多年生草本，全株无毛。叶倒卵形或倒披针形，具不规则波状齿，下面紫色。头状花序直径10mm；总苞狭钟状；花橙黄色至红色。瘦果圆柱形；冠毛白色。为传统山野菜，校内有栽培。

野菊

发表于 2020-06-26 | 分类于 被子植物门（双子叶合瓣花类） |
名称：野菊学名：D. indicum简介：多年生草本，株高3080cm，有香气。茎多分枝，被柔毛。叶互生，卵圆形或卵状椭圆形，羽状深裂。头状花序直径2.54cm，排列为伞房状；花黄色；舌状花雌性；管状花两性。生于山坡、林缘和路边，花序入药。

万寿菊

中华小苦荬

发表于 2020-06-26 | 分类于 被子植物门（双子叶合瓣花类） |
名称：中华小苦荬学名：I. chinense简介：多年生草本，具乳汁。基生叶长椭圆形、倒披针形或舌状，不分裂或羽状裂；茎生叶常24枚，不裂。头状花序含2125朵花；舌状花黄色或白色。瘦果长椭圆形，喙细丝状；冠毛白色。生于山坡、路旁、田野、河边。

生菜

发表于 2020-06-26 | 分类于 被子植物门（双子叶合瓣花类） |
名称：生菜学名：L. sativa var. ramose简介：一年或二年生草本。为莴苣之另一变种，基生叶倒卵形，皱波状，聚成略松散的叶球。常见栽培，叶作蔬菜食用。

刺儿菜

发表于 2020-06-26 | 分类于 被子植物门（双子叶合瓣花类） |
名称：刺儿菜学名：C. setosum简介：多年生草本。叶椭圆或椭圆状披针形，先端锐尖，基部楔形或圆形，全缘或有齿裂，有刺。头状花序单生茎顶，雌雄异株或同株；总苞片多层；管状花紫红色。瘦果椭圆或长卵形；冠毛羽状。生于山坡、河旁或荒地、田间。
