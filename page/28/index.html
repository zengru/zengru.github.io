蚤缀

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：蚤缀学名：Arenaria serpyllifolia简介：一年生草本，株高10~30cm，全株有细毛。茎多数，稍铺散。叶细小，卵圆形，先端尖锐。花小；萼片5；花瓣5，白色；雄蕊10；花柱。蒴果球形。生于路边荒地。

点地梅

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：点地梅学名：Androsace umbellata简介：一年或二年生小草本，全株被细柔毛。叶1030片基生，圆形至心状圆形，边缘具裂齿。伞形花序有415朵花；花小；花冠白色。广布于我国南北各地。

合欢

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：合欢学名：Albizia julibrissin简介：落叶乔木，树皮灰褐色。二回偶数羽状复叶，小叶条形；托叶早落。头状花序排成伞房状；花淡红色；花萼与花冠有短柔毛；雄蕊多数，花丝细柔如丝；雌蕊与雄蕊近等长。荚果条形，扁平。生于山坡、路旁、灌丛中，常栽培作行道树。

金疮小草

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：金疮小草学名：Ajuga decumbens简介：一年或二年生草本，被白色长柔毛。叶匙形或倒卵状披针形，具波状齿。轮伞花序排成穗状；花冠淡蓝紫色至白色，下唇宽大，3裂。小坚果倒卵状三棱形。生于林下、溪边、草坡。

榆叶梅

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |

刺苋

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：刺苋学名：A. spinosus简介：一年生草本。茎多分枝，绿色或紫红色。叶狭卵形或卵状披针形或条状披针形；叶腋生有坚硬的针刺1~2枚。穗状花序顶生，长而直立或稍下垂。常见于村边、路旁和园圃等荒芜地。
