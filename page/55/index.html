萝藦

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：萝藦学名：Metaplexis japonica简介：多年生草质藤本，具乳汁。叶对生，卵状心形，长5~12cm，下面粉绿色；叶柄顶端丛生腺体。总状聚伞花序具长总花梗；花冠白色，内面被柔毛。瞢荚果角状；种子具白色绢质种毛。生于荒地、河边、灌木丛中。

牛繁缕

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：牛繁缕学名：Malachium aquaticum简介：一年生草本，常伏生地面。叶卵形或宽卵形，长2~5.5cm，渐尖。花梗细长；萼片5，短于花瓣；花瓣5，白色，2深裂；雄蕊10；花柱5。蒴果5瓣裂。全国广布，生于路旁及较潮湿的草地。

苹

发表于 2020-06-12 | 分类于 蕨类植物门 |
名称：苹学名：Marsilea minuta简介：沼生草本。根状茎匍匐细长而柔软。不育叶具长柄，叶片十字形，对生，小叶倒三角形，叶脉叉状。孢子果斜卵形或圆形，常2~3个丛集，果内有孢子囊群约15个，每个囊群具少数大孢子囊，周围有数个小孢子囊。为稻田有害杂草。

柘树

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：柘树学名：Maclura tricuspidata简介：落叶灌木或小乔木，树皮淡灰色，成不规则薄片状剥落；小枝具硬刺。叶卵形或倒卵形，长3-12cm，全缘或3裂。花序头状，单生或成对腋生。聚花果近球形，熟时橘红色，直径约2.5cm。生于阳光充足的荒地和路旁。

小苜蓿

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：小苜蓿学名：Medicago minima简介：小草本，铺散，有疏柔毛。叶具3小叶，中间小叶倒卵形，具锯齿；托叶斜卵形。花聚成头状的总状花序；花萼钟状，深裂；花冠黄色；雄蕊10，二体。荚果盘曲成球，具3列钩状长刺。喜生荒坡、沙地，为家畜喜好的牧草，亦可作绿肥。

垂丝海棠

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：垂丝海棠学名：M. halliana简介：落叶小乔木，树冠开展，小枝细弱。叶卵形至长椭圆形，长3.58cm，有圆钝细锯齿。伞房花序有花46朵，花梗细弱，紫色，长24cm；花粉红色，直径33.5cm，下垂。梨果梨形或倒卵形，直径5~6mm，红色。为常见的园林观赏植物。

天蓝苜蓿

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：天蓝苜蓿学名：M. lupulina简介：小草本，有疏毛。小叶3，倒卵形至菱形。花序密集成头状；花冠黄色，略长于花萼。荚果弯曲成肾形，无刺，有疏柔毛。适生于干燥地区，可作绿肥及牧草。

泽珍珠菜

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：泽珍珠菜学名：Lysimachia candida简介：一年生或二年生草本，高15~30cm。基生叶匙形或倒披针形，具带狭翅的长柄；茎生叶常互生，倒卵形、倒披针形或条形，下延至叶柄成狭翅。总状花序顶生；花冠白色。蒴果球形。生于山坡、路旁潮湿处及水田埂上。

北美独行菜

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：北美独行菜学名：Lepidium virginicum简介：一年生或二年生草本。茎直立，上部分枝。基生叶倒披针形，羽状分裂；茎生叶倒披针形或线形。总状花序顶生；花小，白色。短角果近圆形，顶端微缺。原产美洲，为外来入侵种，对农作物有危害。

枫香树

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：枫香树学名：Liquidambar formosana简介：落叶乔木，高达30m，树皮灰褐色。叶阔卵形，掌状3裂，中裂片先端尾状渐尖，叶缘有锯齿；托叶线形。雄花序短穗状；雌花序头状。果序圆球形，木质，具宿存花柱及刺状萼齿。分布于秦岭、淮河以南，校园内有野生，是“枫园”的标志性植物。
