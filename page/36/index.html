半边莲

发表于 2020-06-19 | 分类于 被子植物门（双子叶合瓣花类） |
名称：半边莲学名：Lobelia chinensis简介：多年生草本，株高10~30cm。叶条形或条状披针形。花两侧对称；花冠淡红色，5裂，偏向一侧；花药聚合。生于田边或较潮湿处。

苦瓜

发表于 2020-06-19 | 分类于 被子植物门（双子叶合瓣花类） |
名称：苦瓜学名：Momordica charantia简介：草质藤本；卷须不分叉。叶形或近圆形，5~7深裂，裂片具齿或再分裂。雌雄同株；花萼裂片卵状披针形；花冠黄色，5深裂；花药初时连合而后分离；子房密生疣状突起。果纺锤形或长筒形，熟时裂开。果实作蔬菜。

西瓜

发表于 2020-06-19 | 分类于 被子植物门（双子叶合瓣花类） |
名称：西瓜学名：Citrullus lanatus简介：草质藤本；卷须分2叉。叶3深裂，裂片又羽状或二回羽状裂。雌雄同株；花冠淡黄色；花药为（2）+（2）+1型。果大型，近球状或椭圆状。果为夏日水果，校园内偶见栽培。

小叶猪殃殃

发表于 2020-06-19 | 分类于 被子植物门（双子叶合瓣花类） |
名称：小叶猪殃殃学名：G. trifidum简介：多年生草本。茎纤细，丛生，具4棱。叶小，45片轮生，倒披针形或狭椭圆形，长58mm。花小，白色，3基数。果小，光滑。常生于稍潮湿处。

接骨草

发表于 2020-06-19 | 分类于 被子植物门（双子叶合瓣花类） |
名称：接骨草学名：Sambucus chinensis简介：高大草本至半灌木。羽状复叶，小叶5~9，披针形，具锯齿。复伞房状花序，具不孕花变成的杯状腺体；花白色。果球形，红色。生于林下、沟边或山坡草丛。

荚蒾

发表于 2020-06-19 | 分类于 被子植物门（双子叶合瓣花类） |

丝瓜

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：丝瓜学名：Luffa cylindrica简介：草质藤本；卷须分2~4叉。叶近圆形，掌状5裂，有小锯齿。雌雄同株；花冠黄色；雄蕊5，花药分离。果圆柱状，成熟时干燥。果幼嫩时作蔬菜，成熟果内的网状纤维即“丝瓜络”，入药。

黄瓜

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：黄瓜学名：Cucumis sativus简介：草质藤本，被短刚毛；卷须不分叉。叶宽心状卵形，3~5浅裂，有小锯齿。雌雄同株；花萼裂片钻形；花冠黄色，裂片矩圆形，急尖；花药连合为（2）+（2）+1型。瓠果圆柱状，常有具刺尖的瘤状凸起。普遍栽培，果作蔬菜。

冬瓜

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：冬瓜学名：Benincasa hispida简介：草质藤本；卷须常分为23叉。叶肾状近圆形，57浅裂，有小锯齿，两面有硬毛。雌雄同株；花萼裂片有锯齿；花冠黄色；花药连合为（2）+（2）+1型。果实圆柱状，有毛和白粉。果实为夏日常见蔬菜。
