三角叶酢浆草

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：三角叶酢浆草学名：O. obtriangulata简介：多年生草本，无地上茎。小叶3，广倒三角形，深紫红色。伞形花序，具花10~12朵。花冠白色至粉红色。原产美洲热带地区，校园内偶见栽培。

红花酢浆草

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：红花酢浆草学名：O. corymbosa简介：多年生草本，无地上茎。小叶3，阔倒卵形，长约3.5cm。伞形花序具花5~10朵；花冠淡紫红色。蒴果短条形，角果状。原产美洲热带地区，各地庭院栽培。

四籽野豌豆

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：四籽野豌豆学名：V. tetrasperma简介：一年生纤弱草本。花小，淡蓝紫色（a）。荚果条状椭圆形，长约10mm，常有种子4粒（b）。生于田边、荒地。

蚕豆

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：蚕豆学名：Viciafaba简介：一年或二年生草本。茎略呈四棱。小叶2~6，椭圆形；托叶半箭头形。花冠白色带红具蓝紫色斑纹。荚果肥厚；种子椭圆形，略扁。原产西南亚和北非，相传由张骞自西域引入，为粮食和蔬菜作物。

豌豆

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：豌豆学名：Pisum sativum简介：一年生草本。羽状复叶，小叶1~3对，叶轴顶端为分枝卷须；托叶大，卵形。花冠白色或淡红色；雄蕊为（9）+1型二体。荚果长椭圆形；种子球形。各地栽培，种子与幼荚可食。

黄檀

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：黄檀学名：Dalbergia hupeana简介：乔木；树皮灰色。羽状复叶，小叶9~11，矩圆形或宽椭圆形。圆锥花序；花淡紫色或白色。荚果长圆形，扁平。为优良用材树种。

长萼鸡眼草

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：长萼鸡眼草学名：Kummerowia stipulacea简介：一年生草本，茎平伏。小叶3，倒卵形或椭圆形，侧脉多而密；托叶卵形。花腋生；花冠上部暗紫色。荚果卵形。生于山坡草地上。

芭蕉

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：芭蕉学名：Musa basjoo简介：多年生草本，高46m。叶矩圆形，长达3m，先端钝圆。花序顶生，每段着生小花1020朵，分2层排列；大苞片佛焰苞状，常红褐色；下部为雌性花，中部为中性花，上部为雄性花。浆果长棱形，有时有4~5棱。校园中有栽培。

菝葜

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：菝葜学名：Smilax china简介：攀援灌木。根状茎肥厚、坚硬。枝条常疏生刺。叶薄革质或纸质，宽卵形或圆形，长3~10cm。花单性同株，绿黄色，多排成伞形花序；花被片6，两轮；雌花具6枚退化雄蕊。浆果球形，熟时红色。生于山坡、林下和灌丛中。

朱顶红

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：朱顶红学名：Hippeastrum rutilum简介：多年生草本。鳞茎近球形，直径510cm。叶68枚，排列为2列，带状，长1560cm，先端渐尖。总花梗中空，被白粉，顶端着生24花；花大，直径达20cm；花被6，大红、粉红或橙红色，常有白色条斑。原产秘鲁和巴西，各地庭院栽培。
