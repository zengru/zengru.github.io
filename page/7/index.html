大豆

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：大豆学名：Glycine max简介：一年生草本，密生黄褐色长硬毛。小叶3，卵状菱形。总状花序腋生；花冠小，白色或淡紫色。荚果长圆形，略弯，密生长硬毛；种子2-5，卵形至近球形。为重要优良作物，校内家属区有栽培。

救荒野豌豆

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：救荒野豌豆学名：V.sativa简介：一年或二年生草本。羽状复叶有卷须，小叶816，长椭圆形或倒卵形，先端截形。荚果条形，长2.54.5cm；种子圆球形。遍布全国各省区，嫩茎叶可作野菜。

四季海棠

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：四季海棠学名：Begonia semperflorens简介：肉质草本。叶卵形或宽卵形，长5-8cm，基部略偏斜，有锯齿和睫毛。雄花较大，花被片4；雌花稍小，花被片5。蒴果绿色，有带红色的翅。各地栽培。

三色堇

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：三色堇学名：V. tricolor简介：草本。地上茎直立或倾斜。基生叶长卵形或披针形，茎生叶卵形或长圆状披针形，具圆钝齿。花单生叶腋；每朵的花冠常有多种颜色，假面状，距短而钝。各地栽培，是一种常见的草花。

紫花地丁

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：紫花地丁学名：Viola philippica简介：多年生草本，无地上茎，高4~14cm。叶基生，长圆形、狭卵形至狭卵状披针形，先端圆钝，边缘具圆齿。花单生，苞片线形，生于花梗中部以下；花冠紫堇色或紫色，距细管状，微向上弯或直。蒴果椭圆形。生于林下、山坡、草丛。

厚皮香

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：厚皮香学名：Ternstroemia gymnanthera简介：常绿小乔木或灌木。叶革质，矩圆状倒卵形，基部渐窄而下延。花淡黄色；萼片、花瓣各5；雄蕊多数；子房2~3室。果为干燥的浆果状，萼片宿存。校园中有栽培。

茶

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：茶学名：C. sinensis简介：常绿灌木或小乔木。叶椭圆状披针形至倒卵状披针形，长510cm，有短锯齿。花白色，14朵成腋生聚伞花序。蒴果3室，每室1种子。原产中国，以嫩叶制作茶叶，因工艺不同而有绿茶、红茶、乌龙茶等产品。
