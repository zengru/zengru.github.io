臭牡丹

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：臭牡丹学名：Clerodendrum bungei简介：落叶灌木，有强烈臭气。叶广卵形，长10~20cm，基部心形，边缘有粗锯齿。花蔷薇红色，密集为头状聚伞花序；花冠5裂，下部合生为细管状；雄蕊4。浆果，近于球形，蓝紫色。生于山坡、林缘或沟旁。

日本花柏

发表于 2020-06-12 | 分类于 裸子植物门 |
名称：日本花柏学名：Chamaecyparis pisifera简介：常绿乔木，高可达20m；树皮赤褐色。大枝水平开展，小枝扁平，排成一平面，鳞叶叠瓦状密生，叶面暗绿色。雌雄异株或同株。球果球形；种子三角状卵形，两侧有宽翅。原产日本，我国各地常见栽培。

贴梗海棠

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：贴梗海棠学名：Chaenomeles speciosa简介：落叶灌木，具刺。叶卵形至椭圆形，稀长椭圆形，长39cm；托叶大，肾形或半圆形。花先叶开放，35朵簇生，直径3~5cm，猩红色，稀淡红色或白色；花梗短粗或近无梗。果球形或卵球形，芳香。常植于庭院中或盆栽。

天竺桂

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：天竺桂学名：Cinnamomum pedunculatum简介：常绿乔木。叶近对生，卵状长椭圆形，长10~22cm，革质，具离基三出脉。花黄白色，成圆锥花序。果椭圆形或长椭圆形。分布于东南沿海地区，校内常见盆栽。

醉蝶花

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：醉蝶花学名：Cleome spinosa简介：一年生草本。掌状复叶，小叶5~7。总状花序顶生；花瓣4，淡紫色；雄蕊6；雌蕊梗长。蒴果细圆柱形。栽培观赏植物。

紫荆

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：紫荆学名：Cercis chinensis简介：落叶灌木或小乔木，高达6m；树皮褐色或棕色。叶互生，近圆形，长6-14cm，先端急尖，基部深心形。花先叶开放，4-6朵簇生，玫瑰红色；萼筒偏斜，具5齿；花冠假蝶形；雄蕊10，分离。荚果扁平。常见栽培于庭院中。

紫堇

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：紫堇学名：Corydalis edulis简介：一年生草本，株高10~30cm。叶基生或互生，一至二回羽状全裂，最后裂片顶端2-3齿裂。花序总状；花两侧对称；花瓣粉红色，渐变为紫色。蒴果线形。多生于路边、林下、石壁等阴湿处。

粗梗水蕨

发表于 2020-06-12 | 分类于 蕨类植物门 |
名称：粗梗水蕨学名：Ceratopteris pteridoides简介：多年生。叶二型，无毛。不育叶叶柄粗壮；能育叶末回羽裂条形，孢子囊为反卷的叶缘覆盖。孢子球状四面型。生于湖沼、池塘中。

朴树

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：朴树学名：Celtis sinensis简介：落叶乔木，高可达20m。树皮灰褐色，不开裂。叶阔卵形或圆形，长37cm。中上部边缘有锯齿；三出脉。花杂性，同株。核果近球形，熟时红褐色，直径46cm。产长江流域及华南，校园中有自然分布。
