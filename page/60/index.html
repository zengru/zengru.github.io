马蹄金

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：马蹄金学名：Dichondra repens简介：多年生草本。茎细长，匍匐地面，节上生根。叶圆形或肾形，长5~10mm，先端钝圆或微凹，基部心形，形似马啼，全缘。花小，单生于叶腋，黄色；萼片5；花冠钟状，5深裂，裂片长圆状披针形。蒴果近球形。分布于常见以南各省区，常栽培作草坪。

柿

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：柿学名：Diospyros kaki简介：落叶乔木，树皮鳞片状开裂。叶椭圆状卵形、矩圆状卵形或倒卵形，长6~18cm，，下面淡绿色，有褐色柔毛。雌雄异株或同株；花萼4深裂；花冠白色，4裂；雌花中有8枚退化雄蕊。浆果卵圆形或扁球形，橙黄色或鲜黄色，花萼宿存且增大。全国各地普遍栽培。

石竹

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：石竹学名：Dianthus chinensis简介：多年生草本，光滑无毛。叶线状披针形，宽35mm。花单生或35朵成聚伞花序；苞片4~6，宽卵形；萼圆筒形，先端5裂；花瓣5，鲜红色、白色或粉红色，先端浅裂成牙齿状。蒴果长圆形。生于山坡草丛，各地庭院多有栽培。

蚊母树

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：蚊母树学名：Distylium racemosum简介：常绿灌木或小乔木，小枝和芽有垢状鳞毛。叶革质，椭圆形或倒卵形，长37cm，全缘。总状花序长2cm；苞片披针形；萼筒极短，萼齿大小不等；无瓣；雄蕊56；子房上位，有星状毛，花柱2。蒴果卵圆形，密生星状毛。校园内有栽培。

黑足鳞毛蕨

发表于 2020-06-12 | 分类于 蕨类植物门 |
名称：黑足鳞毛蕨学名：D. fuscipes简介：叶簇生；叶柄基部黑色，密被鳞片。叶片卵状披针形或三角状卵形，二回羽状。叶轴和羽轴具泡状鳞片。孢子囊群略靠近小羽片中脉着生。生 疏林下或灌丛中。

渐尖毛蕨

发表于 2020-06-12 | 分类于 蕨类植物门 |
名称：渐尖毛蕨学名：Cyclosorus acuminatus简介：多年生。根状茎长而横走。叶远生；叶柄深禾秆色；叶片阔披针形，二回羽裂，羽片披针形，渐尖头，羽裂1/22/3，裂片又侧脉79对。孢子囊群生裂片的侧脉上部，囊群盖圆肾形，有密柔毛。生于田边、路旁、山谷。

君迁子

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：君迁子学名：D.lotus简介：落叶乔木，高达14m。树皮光滑，幼枝灰绿色。叶椭圆形至矩圆形，长6~12cm，下面近白色。花单性异株，簇生于叶腋；花萼3裂。浆果球形，熟时蓝黑色。生于山坡、山谷。

宁波溲疏

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：宁波溲疏学名：Deutzia ningpoensis简介：灌木；小枝疏生星状毛。叶对生，狭卵形或披针形。花序圆锥状；萼5裂；花瓣5，白色；雄蕊10；子房下位，花柱3~4。蒴果近球形。校园内有栽培。

头石竹

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：头石竹学名：D. barbatus var. asiaticus简介：多年生草本。叶披针形或椭圆状披针形，宽1.5~2.5cm，先端渐尖。聚伞花序聚成紧密的头状；花梗短；苞片与萼筒等长；花瓣5，先端有细齿，紫色、绯红色或白色，或有斑纹。各地庭院多有栽培。
