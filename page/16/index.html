凤尾丝兰

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：凤尾丝兰学名：Yucca gloriosa简介：多年生常绿木本。叶剑形、坚硬，密生成莲座状，长4060cm，顶端为坚硬的刺。花葶高12m；大型圆锥花序具多花；花白色至乳黄色，下垂，钟形；花被6片，卵状菱形。常见栽培，多用于配植花坛。

唐菖蒲

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：唐菖蒲学名：Gladiolus gandavensis简介：多年生草本。鳞茎扁圆形。叶剑形，2列，长达60cm。穗状花序顶生，长达30cm；花红黄色、白色或淡红色；花被筒漏斗状，上部6裂，内轮3片较大；雄蕊3；子房3室。蒴果矩圆形至倒卵形。各地庭院栽培。

石蒜

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：石蒜学名：Lycoris radiata简介：多年生草本；鳞茎近球形。叶条形或带形，宽1~2cm。花葶在叶前抽出，高约30cm；伞形花序；花鲜红色或具白色边缘，长约7.5cm；花被片6，边缘皱缩，向后反卷；雄蕊6；子房下位，3室。校内栽培观赏。

绶草

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：绶草学名：Spiranthes sinensis简介：陆生，具肉质根。茎直立，高1550cm。叶24枚，条状倒披针形。花序似穗状，花多数，白色或淡红色，螺旋状排列；唇瓣近矩圆形，长4~5mm，顶端极钝，中部之上具强烈皱波状啮齿。广布全国各地，生于山坡林下或草地。

名称：水仙学名：Narcissus tazetta简介：多年生草本。鳞茎卵圆形。叶直立而扁平，宽11.8cm，顶端钝。花葶中空，约与叶等长；总苞片佛焰苞；伞形花序含410朵花；花被高脚碟状，裂片6，倒卵形，白色；副花冠浅杯状，淡黄色；雄蕊6；柱头3裂。蒴果，室背开裂。原产福建和浙江，常见盆栽。 ...

君子兰

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：君子兰学名：Clivia miniata简介：多年生草本。叶着生在短缩茎上，互生，排成2裂，带状，革质，长达45cm。伞形花序顶生，有7~30朵花；花直立，具花梗；花冠漏斗状，花被片6，倒卵形，黄色或橘红色。原产非洲南部，常见栽培观赏，有很多名贵品种。

春兰

发表于 2020-06-28 | 分类于 被子植物门（单子叶类） |
名称：春兰学名：Cymbidium goeringii简介：陆生，假鳞茎集生成丛。叶46枚丛生，狭带形，长2040cm，具细锯齿。花葶直立，远比叶短；花单生，少为2朵，浅黄绿色，清香；唇瓣不明显3裂，比花瓣短，浅黄色带紫褐色斑点。珞珈山有自然分布。

蕙兰

发表于 2020-06-27 | 分类于 被子植物门（单子叶类） |
名称：蕙兰学名：C. faberi简介：陆生。叶79枚丛生，条形，长2580cm。中下部常对褶，边缘有细齿。花葶直立，高3080cm，总状花序具612花；花浅黄绿色，清香；花瓣略小于萼片，唇瓣不明显3裂，有紫色斑点。野生于林下阴湿处，各地广为栽培，品种极多。

洋葱

发表于 2020-06-27 | 分类于 被子植物门（单子叶类） |
名称：洋葱学名：A. cepa简介：多年生草本。叶中空。花葶粗壮，高可达1m。伞形花序球形，多花，密集；花梗为花被的数倍长；花被片6；花丝比花被片长，约1/5合生并与花被贴生。原产亚洲细胞，世界各地栽培，鳞茎球形或扁球形，粗大，供食用。

郁金香

发表于 2020-06-27 | 分类于 被子植物门（单子叶类） |
名称：郁金香学名：Tulipa gesneriana简介：多年生草本。鳞茎扁圆锥形，具褐色皮膜。叶长椭圆状披针形，长10~25cm。花单生茎顶，大形直立；花被片6，分离；雄蕊6；子房椭圆形，柱头3裂。原产地中海沿岸及中亚细亚、伊朗、士耳其、前苏联南部及我国西藏、新疆等地，各地栽培，花色丰富。
