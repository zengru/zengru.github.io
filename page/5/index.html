马松子

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：马松子学名：Melochia corchorifolia简介：半灌木状草本，散生星状柔毛。叶卵形、狭卵形或三角状披针形，边缘具小牙齿。头状花序；花冠白色至淡紫色；花丝大部合生成管。分布于长江以南各省区。

陆地棉

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：陆地棉学名：Gossypium hirsutum简介：一年生草本，株高达1.5m。叶互生，宽卵形，掌状3（5）裂。花单生；小苞片3；花冠白色或淡黄色。蒴果卵形，长3.55cm；种子具长棉毛。原产中美洲，为重要经济作物。

槭葵

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：槭葵学名：H. coccineus简介：多年生草本，高13m，光滑。叶掌状57深裂，裂片线状披针形，有不规则锯齿。花大，深红色。原产北美洲，校内偶见栽培。

朱槿

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：朱槿学名：H. rosa-sinensis简介：常绿灌木，株高1-3m。叶阔卵形或狭卵形，长49cm，具粗齿或缺刻；托叶线形。花单生于上部叶腋，常下垂；小苞片67；萼5裂；花冠漏斗状，直径610cm；雄蕊柱长48cm；花柱5。蒴果卵形。常见栽培观赏，花色多样。

苘麻

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：苘麻学名：A. theophrasti简介：一年生草本，被柔毛或星状毛。叶近圆形，渐尖，边缘有疏密不等的粗齿。花黄色。蒴果直径约2cm，分果爿13-20。世界各地广布，农家栽培，为麻类作物。

金铃花

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：金铃花学名：Abutilon striatum简介：常绿灌木。叶掌状35深裂，长58cm，裂片卵状，先端长渐尖，边缘具锯齿或粗齿。花单生叶腋，下垂，钟形，桔黄色，具深红色条纹。原产南美洲，校内偶见栽培。

冬葵

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：冬葵学名：Malva crispa简介：二年生草本，株高60-90cm。叶肾形至圆形，掌状57浅裂，两面常被糙伏毛。花小，淡红色，常丛生叶腋间；小苞片3；萼5齿裂；花瓣5；子房1011室。果扁圆形，熟时心皮彼此分离并与中轴脱离。全国广布，嫩苗可作蔬菜。

扁担杆

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：扁担杆学名：Grewia biloba简介：落叶灌木或小乔木。叶菱状卵形，基出脉3条。聚伞花序；花黄绿色；萼片、花瓣各5。核果橙红色。生于山坡灌丛。

五叶地锦

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：五叶地锦学名：P. quinquefolia简介：木质藤本，卷须总状59分枝。掌状复叶，小叶5，倒卵圆形、倒卵椭圆形或外侧小叶椭圆形，长515cm。花序假顶生形成主轴明显的圆锥状多歧聚伞花序。分布于东北至华南各省区。

乌蔹莓

发表于 2020-06-28 | 分类于 被子植物门（双子叶离瓣花类） |
名称：乌蔹莓学名：Cayratia japonica简介：多年生蔓生草本。鸟足状复叶，小叶5，椭圆状卵形。聚伞花序与叶对生；花黄绿色；萼杯状；花瓣4；雄蕊4，对瓣。浆果圆卵形，熟时黑色。
