梓树

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：梓树学名：Catalpa ovata简介：落叶乔木。叶对生或轮生，宽卵形或近圆形，常35浅裂。花多数，成圆锥花序；花冠淡黄色，内有黄色线纹和紫色斑点。蒴果线形，长2030cm。分布于长江流域及以北地区，为速生树种。

芝麻

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：芝麻学名：Sesamum indicum简介：一年生草本，被短柔毛。茎四棱形。叶卵形、矩圆形或披针形，下部叶3浅裂。花腋生；花冠白色或有彩晕，裂片圆形。蒴果椭圆形，多4棱或6、8棱，纵裂；种子多数。各地广泛种植，种子榨油或供食用。

厚萼凌霄

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：厚萼凌霄学名：Campsis radicans简介：落叶木质藤本。羽状复叶，小叶7~11。聚伞花序；萼筒钟状；花冠漏斗状钟形，橘红色，5裂；雄蕊2长2短，另1枚退化。蒴果豆荚状。校内多栽培，攀附于藤萝架上。

簇生椒

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：簇生椒学名：C. annuum var. fasciculatum简介：叶和浆果簇生于枝端。果梗直立，浆果长指状，顶端渐尖。为辣椒的变种。

匍茎通泉草

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：匍茎通泉草学名：Mazus miquelii简介：多年生草本，匍匐或上升。基生叶匙形；茎生叶匙形或近圆形，有粗齿。总状花序顶生；花冠紫色或白色而有紫斑，下唇3裂片突出，倒卵圆形。蒴果球形。生于潮湿的路边、荒地。

弹刀子草

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：弹刀子草学名：M. stachydifolius简介：多年生草本，被白色长柔毛。叶匙形或长矩圆形，具不规则锯齿。花冠紫色，下唇3裂片突出。蒴果卵球形形。生于湿润的路旁、草坡和林缘。

金鱼草

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：金鱼草学名：Antirrhinum majus简介：多年生直立草本。叶披针形至矩圆状披针形，全缘。总状花序顶生，密被腺毛；花冠颜色多样，基部在前面下延成兜状。原产欧洲南部，各地庭院常见栽培。

蚊母草

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：蚊母草学名：V. peregrina简介：一年生小草本，无毛或有腺毛。下部叶倒披针形，上部叶长矩圆形，全缘或有不明显的齿。总状花序顶生，苞片比叶略小；萼4裂，裂片狭披针形，先端钝；花冠白色或略带淡红色，4深裂；雄蕊2。蒴果倒心形，常因昆虫寄生而肿胀。生于潮湿荒地。

水苦荬

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：水苦荬学名：V. undulata简介：一年生草本。茎、花序轴、花梗、花萼和蒴果有大头针状腺毛。叶卵状矩圆形或条状披针形，基部常半抱茎。花序宽达1.5cm；花冠淡蓝紫色或白色；花柱较短。蒴果卵圆形，顶端微凹。生于水沟边或潮湿处。

珊瑚樱

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：珊瑚樱学名：S. pseudocapsicum简介：直立小灌木，全体无毛。叶狭矩圆形或披针形。花常单生；花冠白色。浆果橙红色直径1~1.5cm。原产南美洲，我国各地栽培观赏。
