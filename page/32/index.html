泥胡菜

发表于 2020-06-26 | 分类于 被子植物门（双子叶合瓣花类） |
名称：泥胡菜学名：Hemistepta lyrata简介：二年生草本。基生叶倒披针形或倒披针状椭圆形，提琴状羽裂；中部叶椭圆形；羽裂；上部叶条状披针形至条形。总苞片背面具鸡冠状附属体；花紫色。瘦果圆柱形；冠毛白色，羽状。分布全国各地，常见于路边荒地。

野艾蒿

发表于 2020-06-26 | 分类于 被子植物门（双子叶合瓣花类） |
名称：野艾蒿学名：A. lavandulaefolia简介：多年生草本。下部二回羽状分裂，上部叶羽状深裂至全裂，裂片线状披针形，下面密生灰白色短毛。头状花序直径约2.5cm；总苞片4层。生于山坡草地或沟边。

艾蒿

发表于 2020-06-26 | 分类于 被子植物门（双子叶合瓣花类） |
名称：艾蒿学名：Artemisia argyi简介：多年生草本。中部叶长59cm，羽状深裂或浅裂，侧裂片约2对，中裂片又常3裂，下面被密茸毛；上部叶渐小。头状花序多数，直径23mm；总苞片4~5层；花带红色，外层雌性，内层两性。分布几遍全国，生于山坡、荒地、路旁。

狼把草

发表于 2020-06-26 | 分类于 被子植物门（双子叶合瓣花类） |
名称：狼把草学名：B. tripartita简介：一年生草本，株高30150cm。叶对生；中部叶常羽状35裂；上部叶3深裂或不裂。头状花序直径13cm；总苞片多数，外层倒披针形，叶状，长14cm；花黄色，全为两性筒状花。瘦果扁平；冠毛芒状，2枚。生于旷野、水边。

千里光

发表于 2020-06-26 | 分类于 被子植物门（双子叶合瓣花类） |
名称：千里光学名：Senecio scandens简介：多年生草本，茎曲折呈蔓生状。叶互生，卵状披针形或椭圆状三角形，长7~10cm，先端渐尖。头状花序排成圆锥状伞房花丛；总苞圆筒形；舌状花黄色，先端3齿裂；管状花，两性。瘦果圆筒形，有细毛；冠毛白色。生长在山坡、路旁、林缘。

鳢肠
