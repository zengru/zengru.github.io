狗牙根

发表于 2020-06-27 | 分类于 被子植物门（单子叶类） |
名称：狗牙根学名：Cynodon dactylon简介：多年生草本，具根状茎或匍匐茎。叶舌具小纤毛；叶片宽13mm。穗状花序36枚指状排列；小穗排列于穗轴一侧，长2~2.5mm，含1小花；颖近等长，短于外稃；外稃具3脉。生旷野、路边及草地，可用于配植草坪。

知风草

发表于 2020-06-27 | 分类于 被子植物门（单子叶类） |
名称：知风草学名：Eragrostis ferruginea简介：多年生草本。秆高2575cm。叶鞘压扁；叶舌退化成短毛；叶片宽46mm。圆锥花序长2030cm，开展，基部常包于鞘内；小穗带紫黑色，长510mm，含7~12花；外稃具3脉，长3mm，自下而上脱落。生于路边、田野。

菵草

发表于 2020-06-27 | 分类于 被子植物门（单子叶类） |
名称：菵草学名：Beckmannia syzigachne简介：一年生草本，高4580cm。叶舌膜质；叶片长615cm；圆锥花序长10~25cm；小穗常单生，压扁，脱节于颖下；颖半圆形，泡状膨大，稍革质；内外稃等长，膜质，具长约0.5mm的芒尖。常见于水边湿地。

台湾剪股颖

发表于 2020-06-27 | 分类于 被子植物门（单子叶类） |
名称：台湾剪股颖学名：Agrostis canina简介：多年生草本。秆丛生，高3060cm。叶鞘无毛；叶舌先端平，常破裂；叶片线形，微粗糙。圆锥花序，疏松开展，每节24枚分枝；小穗长2~2.5mm，老后变黄紫色。两颖近相等；外稃具5脉，芒细，直或稍扭曲。见于珞珈山山坡和路边草丛。

棒头草

发表于 2020-06-27 | 分类于 被子植物门（单子叶类） |
名称：棒头草学名：Polypogon fugax简介：一年生草本。高1575cm。叶鞘平滑，叶舌膜质；叶片长515cm。圆锥花序长圆形，较疏松；小穗长2mm；颖近等长，粗糙，中脉延伸成长1~3mm的细芒；外稃长约1mm，顶端具易脱落的细芒。生于田边或潮湿处。

看麦娘

发表于 2020-06-27 | 分类于 被子植物门（单子叶类） |
名称：看麦娘学名：Alopecurus aequalis简介：越年生草本。高1540cm。叶鞘短于节间；叶舌膜质；叶片宽25mm。圆锥花序狭圆柱形，长2~7cm；小穗含1花，脱节颖下；颖相等，基部合生；芒细弱，约于稃体下部1/4处伸出。生于田边或潮湿草地。

野燕麦

发表于 2020-06-27 | 分类于 被子植物门（单子叶类） |
名称：野燕麦学名：Avena fatua简介：一年生或越年生草本，高达150cm。叶舌膜质；叶片疏被短柔毛。圆锥花序开展；小穗长1825mm，含23小花，颖近等长；外稃坚硬，中部稍下方有一长4cm具曲膝的芒。颖果纺锤形，长6~8mm，有毛。为荒野或田间常见杂草。

黑麦草

发表于 2020-06-27 | 分类于 被子植物门（单子叶类） |
名称：黑麦草学名：Lolium perenne简介：多年生草本，秆丛生，高5010cm。叶鞘常短于节间，叶舌短小。叶长1025cm，被微毛。穗状花序长1030cm；小穗含511小花，颖短于小穗；外稃披针形，常无芒，内稃与外稃等长。校内栽培作草坪。

疏花雀麦

发表于 2020-06-27 | 分类于 被子植物门（单子叶类） |
名称：疏花雀麦学名：B. remotiflorus简介：一多年生草本，具短根状茎。叶片宽48mm。圆锥花序疏松、开展，熟时下垂；小穗暗绿色，宽34mm，疏生510小花；颖狭披针形。颖果长810mm，贴生于稃内。生于山坡、林缘、路旁和草地。

疣粒野生稻

发表于 2020-06-27 | 分类于 被子植物门（单子叶类） |
名称：疣粒野生稻学名：O. granulata简介：多年生水生草本；高3070cm。叶叶舌长12mm，叶耳明显；叶片线状披针形。圆锥花序疏生小穗；小穗长约6mm，孕性外稃无芒，具小疣点。校内栽培作实验材料。
