重瓣郁李

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：重瓣郁李学名：C. japonica var. kerii简介：落叶灌木。叶背面无毛。花半重瓣，粉红色；花梗短。为郁李之变种，产我国南方诸省，具较高观赏价值。

凤尾鸡冠花

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：凤尾鸡冠花学名：C. cristata cv. ‘Pyramidalis’简介：一年生草本。为鸡冠花的栽培变种。花序多分枝，呈圆锥花序状，似火炬。

构树

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：构树学名：Broussonetia papyrifera简介：落叶乔木，具乳汁；树皮平滑，浅灰色或灰褐色。单叶互生，有时近对生，卵圆至阔卵形，有粗齿，或3~5深裂，有厚柔毛；托叶卵状长圆形。雄花成柔荑花序，下垂。聚合果球形，熟时橙红色或鲜红色。黄河、长江、珠江流域广布，校园中常见野生。

光叶子花

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：光叶子花学名：Bougainvillea glabra简介：攀援灌木，具腋生直刺。叶卵形或卵状披针形，长5~10cm。花常3朵簇生在苞内；苞片叶状，暗红色或紫色，椭圆形。原产巴西，各地栽培观赏。

红花高盆樱

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：红花高盆樱学名：C. cerasoides var. rubea简介：落叶乔木。叶倒卵状长椭圆形至椭圆状卵形，叶柄近端处有2~3个腺体。花粉红至深红色，单瓣或重瓣，略下垂。产于云南和西藏，校内有引种，2月底开花。

鸡冠花

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：鸡冠花学名：C. cristata简介：一年生草本。茎直立，略呈红色或紫红色。叶卵状长圆形或卵状披针形。花序顶生，扁平鸡冠状；花色多样而艳丽，有紫、红、淡红、黄或杂色。各地栽培观赏，也有半野生状态。

卷心菜

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：卷心菜学名：Brassica oleracea var. capitata简介：二年生草本，被粉霜。基生叶多层，结成扁球形，直径达30cm。春季抽出总状花序，花黄色。原产于地中海沿岸，各地广泛栽培，别称“包菜”等。

柔弱斑种草

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：柔弱斑种草学名：Bothriospermum tenellum简介：一年生草本，有糙伏毛。叶狭椭圆形或矩圆状椭圆形。花序狭长；苞片椭圆形或狭卵形；花萼5裂至近基部；花冠淡蓝色，直径约2mm，喉部有5个附属物；雄蕊5；子房4裂。小坚果4，肾形，密生小疣状突起。生于地边、路旁。

樟

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：樟学名：C. camphora简介：常绿乔木，有香气。叶长圆状卵形至卵形，长59cm，薄革质，具离基三出脉。花序圆锥状；花小，两性；花被6片；雄蕊9；子房卵圆形。果球形，直径46mm，熟时蓝黑色。分布于华中、华东至华南，是珞珈山植被的主要成分，木材优良。

落葵

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：落葵学名：Basella rubra简介：一年生缠绕草本，肉质多汁；茎长可达数米。叶卵形或近圆形，长49cm，全缘，顶端钝圆。穗状花序腋生；小苞片2，萼状，宿存；萼片5，淡红色或淡紫色；雄蕊5；花柱3裂。果球形，直径56mm。各地栽培，嫩叶作蔬菜食用。
