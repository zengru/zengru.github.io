柳杉

发表于 2020-06-12 | 分类于 裸子植物门 |
名称：柳杉学名：Cryptomeria fortunei简介：常绿乔木，高达 40m。树皮红棕色，裂成长条片脱落。小枝常下垂。叶钻形，略内曲，长 1~1.5 cm。雄球花长椭圆形，集生于小枝上部。球果圆球形或扁球形；能育种鳞有 2 粒种子，边缘有窄翅。中国特有种，分布于长江流域以南。

青冈栎

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：青冈栎学名：Cyclobalanopsis glauca简介：常绿乔木。叶互生，革质，椭圆形或倒卵状椭圆形，长8~10cm，中部以上边缘具齿。壳斗杯状，有 6-8 层鳞片环；坚果椭球形。为亚热带树种，珞珈山有分布。

苏铁

发表于 2020-06-12 | 分类于 裸子植物门 |
名称：苏铁学名：Cycas revoluta简介：常绿乔木，高1~4m。营养叶羽状，小叶线形，坚硬，有光泽。雌雄异株；小孢子叶球圆柱形，黄色；大孢子叶聚生成扁球形。种子大，熟时红褐色或橘红色。分布于福建、广东，各地栽培观赏，华中地区罕见“开花”。

菟丝子

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：菟丝子学名：Cuscuta chinensis简介：一年生寄生草本。茎细，黄色，无叶。花簇生；花萼5裂；花冠白色，5裂，裂片向外反曲；雄蕊5；鳞片5，边缘流苏状；花柱2。蒴果近球形。缠绕于草本植物上。

仙客来

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：仙客来学名：Cyclamen persicum简介：多年生草本，具球形或扁球形块茎。叶心形、卵形或肾形，有细锯齿，叶面有白色或灰色晕斑。花单生于花葶上，有白色、粉红、红色等颜色。是著名的冬、春季节温室盆花。

臭荠

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：臭荠学名：Coronopus didymus简介：一年或二年生匍 匐草本，有臭味。叶一 至二回羽状分裂，裂片 5~7 对，条形。总状花 序腋生；花白色，直径 约 1 mm；短角果近半 球形，侧扁。常见的杂 草，生于路边荒地。

臭牡丹

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：臭牡丹学名：Clerodendrum bungei简介：落叶灌木，有强烈臭气。叶广卵形，长10~20cm，基部心形，边缘有粗锯齿。花蔷薇红色，密集为头状聚伞花序；花冠5裂，下部合生为细管状；雄蕊4。浆果，近于球形，蓝紫色。生于山坡、林缘或沟旁。

日本花柏

发表于 2020-06-12 | 分类于 裸子植物门 |
名称：日本花柏学名：Chamaecyparis pisifera简介：常绿乔木，高可达20m；树皮赤褐色。大枝水平开展，小枝扁平，排成一平面，鳞叶叠瓦状密生，叶面暗绿色。雌雄异株或同株。球果球形；种子三角状卵形，两侧有宽翅。原产日本，我国各地常见栽培。
