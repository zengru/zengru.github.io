化香树

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：化香树学名：Platycarya strobilacea简介：落叶小乔木。奇数羽状复叶，互生，小叶 7~23，卵状披针形至椭圆状披针形，边缘有重锯齿。花单性或两性，雌雄同株。花序直立，两性或雄性。果序球果状，卵状椭圆形至椭圆状圆柱形。珞珈山有自然分布。

加杨

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：加杨学名：Populus ×canadensis简介：落叶大乔木，高可达30m。树皮灰褐色。叶三角形或三角状卵形，无毛，边缘具钝齿；叶柄侧扁。雄花有雄蕊 15~25。果序长达 27cm。为杂交种，原产法国，现栽培于我国南北各地。

罗松柏

发表于 2020-06-12 | 分类于 裸子植物门 |
名称：罗松柏学名：Podocarpus macrophyllus简介：常绿乔木。叶条状披针形，螺旋状互生。雌雄异株，偶同株。雄球花柔荑花序状。种子卵形，有黑色假种皮，着生于肉质而膨大的种托上。分布于长江流域以南各省区，多系栽培。

一球悬铃木

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：一球悬铃木学名：Platanus occidentalis简介：落叶乔木；树皮小块状剥落。叶阔卵形，3（5）浅裂；托叶较大。花单性，常46基数。果序单生，稀为2个，直径约3cm。校内偶见栽培。

海桐

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：海桐学名：Pittosporum tobira简介：常绿灌木或小乔木。叶互生，倒卵形或椭圆形，长5~12cm，全缘，厚革质，有光泽。花白色或淡黄色，芳香。蒴果卵球形，三瓣裂；种子红色。校园内常见栽培，树冠常修剪成圆球形。

美洲商陆

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：美洲商陆学名：Phytolacca americana简介：多年生草本，光滑无毛。根肥大。叶椭圆状卵形或披针形。总状花序下垂；花白色；雄蕊10；雌蕊心皮10，合生。浆果扁球形。原产北美洲。

日本五针松

发表于 2020-06-12 | 分类于 裸子植物门 |
名称：日本五针松学名：Pinus parviflora简介：常绿乔木，树冠圆锥形，枝平展。针叶5针一束，微弯曲，长3.5~5.5cm；叶鞘早落。球果卵圆形或卵状椭圆形，几无梗，熟时种鳞张开。种子为不规则倒卵圆形，近褐色，具黑色斑纹。原产日本等地，各地引种栽培。

石楠

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：石楠学名：Photinia serrulata简介：常绿灌木或小乔木。叶革质，长椭圆形或倒卵状椭圆形，长922cm，疏生具腺细锯齿，先端尾尖。复伞房花序顶生；花瓣白色，近圆形。梨果球形，直径56mm，红色。为常见的园林栽培树种。

名称：小天蓝绣球学名：Phlox drummondii简介：一年生草本。叶卵形、矩圆形或披针形，全缘。聚伞花序顶生；苞片和小苞片条形；花冠高脚碟形，有红、白、黄、紫等色。各地庭院栽培。
