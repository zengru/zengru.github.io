婆婆纳

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：婆婆纳学名：Veronica didyma简介：一年生草本，匍匐或上升。叶对生，三角状圆形，钝锯齿。花冠蓝紫色，辐状，直径4~8mm，4裂；雄蕊2。蒴果近肾形，稍扁。生于荒地、路旁。

直立婆婆纳

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：直立婆婆纳学名：V. arvensis简介：小草本；茎直立或上升，高10~30cm。叶圆形或卵圆形，具圆或钝齿，两面具硬毛。总状花序细长；花小，蓝紫色或蓝色（a）。蒴果倒心形，侧扁（b）。生于路边荒地。

白花泡桐

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：白花泡桐学名：Paulownia fortunei简介：落叶大乔木，树皮灰褐色。叶对生，心状卵圆形至心状长卵形，长达20cm，全缘。聚伞圆锥花序顶生；花冠白色，内有紫斑，外被星状绒毛上唇2裂，下唇3裂；雄蕊4，2强。蒴果长达8cm，外果皮硬壳质。分布于长江以南地区，为速生树种。

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：碧冬茄学名：Petunia hybrida简介：一年生或多年生草本，有腺毛。叶卵形。花冠漏斗状，5钝裂；雄蕊5，两两成对，1枚退化。蒴果2瓣裂。原产南美洲，常见栽培于庭院和广场。

孔雀草

名称：孔雀草学名：T. patula简介：一年生草本。叶羽状分裂，裂片线状披针形，边缘有锯齿。头状花序直径3.5~4cm，花序梗顶端稍增粗；总苞长椭圆状；舌状花金黄色或橙色，带有红色斑；管状花花冠黄色。常见庭院栽培。

宝盖草

发表于 2020-06-13 | 分类于 被子植物门（双子叶合瓣花类） |
名称：宝盖草学名：Lamium amplexicaule简介：一年或二年生草本。叶无柄，圆形或肾形，长12cm，两面均疏生伏毛。轮伞花序610花；花冠粉红至紫红色，筒细长，上唇直立，下唇3裂。小坚果倒卵状三棱形。生于林缘、草地，是校园中早春常见的杂草。

蓝猪耳

发表于 2020-06-13 | 分类于 被子植物门（双子叶合瓣花类） |
名称：蓝猪耳学名：Torenia fournieri简介：直立草本，高达30cm。茎四方形。叶卵形或卵状披针形，长3~5cm，有锯齿。花腋生或为顶生总状花序；花冠管青紫色、桃红色、蓝紫色等，下唇中间裂片有黄斑。原产亚洲热带和亚热带地区，为草本花卉。

发表于 2020-06-13 | 分类于 被子植物门（双子叶合瓣花类） |
名称：阿拉伯婆婆纳学名：V. persica简介：一年生草本，铺散状。叶卵形或圆形，具钝齿。花冠蓝色或蓝紫色，直径约1cm，4裂。蒴果肾形。原产西亚和欧洲，是校园中常见的早春开花植物。
