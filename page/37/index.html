水栀子

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：水栀子学名：G. jasminoides var. radicans简介：植株矮小。叶倒披针形，长4~8cm。花亦较小。为子的变种，校园中偶见。

番茄

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：番茄学名：Lycopersicon esculentum简介：一年生或多年生草本；叶为羽状复叶或羽状分裂；花序生于叶腋外；萼片56；花冠黄色，56裂；雄蕊5~6，花药靠合，纵裂；浆果扁球形或近球形，红色或橘黄色。果作蔬菜或水果。

栀子

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：栀子学名：Gardenia jasminoides简介：常绿灌木。叶对生或3叶轮生，倒卵状长椭圆形，长7~13cm，全缘；托叶膜质，合生成鞘。花单生枝顶；花冠白色，裂片5或更多，具浓香；雄蕊6，无花丝；子房1室。果倒卵形或长椭圆形，黄色，常有翅状纵棱6条，顶端有宿萼。常见栽培观赏。

爵床

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：爵床学名：Rostellularia procumbens简介：细弱草本，常有短硬毛。叶椭圆形至椭圆状矩圆形。穗状花序；花小；花冠粉红色，二唇形；雄蕊2。生于荒地草丛中。

车前

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：车前学名：P. asiatica简介：多年生草本，全体光滑或有短毛。叶丛生于根茎顶端，有长柄，广卵形或长圆状卵形，长415cm，全缘或疏生钝齿。花葶长1030cm，花小，绿白色。蒴果卵状圆锥形，近中部周裂。生路边、沟旁、田埂等处。

平车前

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：平车前学名：Plantago depressa简介：一年生草本基生叶直立或平铺，椭圆形、椭圆状披针形至卵状披针形，具疏齿，有柔毛或无毛。穗状花序长4~10cm。蒴果圆锥状，周裂。分布几遍全国。

金苞花

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：金苞花学名：Pachystachys lutea简介：常绿亚灌木。茎节膨大。叶对生，长椭圆形。穗状花序顶生，苞片金黄色；花冠白色。原产美洲热带地区，温室栽培。

辣椒

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：辣椒学名：Capsicum annuum简介：灌木或栽培为一年生草本。叶常为卵状披针形。花单生于叶腋或枝腋；花冠辐状，白色，裂片5~7。浆果俯垂，果皮和胎座间有空腔，熟后红色。原产南美洲，各地栽培，作蔬菜及调味品。

黄花狸藻

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：黄花狸藻学名：Utricularia aurea简介：水生食虫植物。茎浮水，极延长。叶沉水，2回至3回羽状分裂，裂片细条形，基部有近球形的小囊体。花茎伸出；花黄色；花萼2裂；花冠唇形；雄蕊2；子房上位，1室。蒴果圆锥形。生于静水中。

大岩桐

发表于 2020-06-16 | 分类于 被子植物门（双子叶合瓣花类） |
名称：大岩桐学名：Sinningia speciosa简介：多年生草本，株高1525cm，密被白色绒毛。叶肥厚，卵圆形或长椭圆形，有锯齿。花顶生或腋生，大而美丽；花冠钟状，先端浑圆，56浅裂，有粉红、红、紫蓝、白等色。校内偶见栽培。
