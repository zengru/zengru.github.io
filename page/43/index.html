落羽杉

发表于 2020-06-12 | 分类于 裸子植物门 |
名称：落羽杉学名：T. distichum简介：落叶乔木，树皮长条片状脱落，棕色。叶线形，宽约1mm，螺旋状互生，在小枝上扭转成羽状2列。球果圆形或卵圆形，有短梗。原产于美国东南部，长江流域及以南各地广泛栽培。

雀舌草

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：雀舌草学名：Stellaria alsine简介：越年生草本。茎细弱，无毛。叶无柄，矩圆形至卵状披针形，长5~20mm。花序聚伞状或单花腋生；花梗细；萼片5；花瓣5，白色，2深裂几达基部；雄蕊5；花柱2-3。蒴果6裂。生于田间、溪岸或潮湿处。

野茉莉

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：野茉莉学名：Styrax japonicus简介：灌木或乔木。叶椭圆形至卵状椭圆形，长410cm。花白色，58朵生于侧枝顶端，下垂；子房上位。核果卵圆形。生于山坡或沟谷中。

紫丁香

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：紫丁香学名：Syringa oblata简介：灌木或小乔木。叶圆卵形至肾形，全缘。圆锥花序；花冠紫色，筒长1015mm，4裂；蒴果长11.5cm，顶端尖。生于山地或山沟，庭院栽培，花可提取芳香油。

矮雪轮

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：矮雪轮学名：Silene pendula简介：一年或二年生草本。高约30cm。分枝多。叶卵状披针形。聚伞花序；萼筒长而膨大，筒上有紫红色条筋。花瓣倒心形，先端2裂，粉红色；蒴果卵形。栽培品种花色有白色、淡紫色、浅粉红色、玫瑰色等。

秤锤树

发表于 2020-06-12 | 分类于 被子植物门（双子叶合瓣花类） |
名称：秤锤树学名：Sinojackia xylocarpa简介：落叶小乔木。叶椭圆形或椭圆状倒卵形，有细锯齿。花白色，35朵组成总状花序；子房半下位，34室。坚果木质，下垂，栗褐色。珍稀濒危植物，校园中有栽培。

垂盆草

发表于 2020-06-12 | 分类于 被子植物门（双子叶离瓣花类） |
名称：垂盆草学名：Sedum sarmentosum简介：多年生肉质草本，不育枝匍匐。叶3片轮生，倒披针形至长圆形，长15~25mm，全缘。花序聚伞状，顶生；花无梗；萼片5，披针形至长圆形；花瓣5，黄色，披针形；雄蕊10，较花瓣短；心皮5，略叉开。生于山坡岩石上或沟边、路旁潮湿处。

伏地卷柏

发表于 2020-06-12 | 分类于 蕨类植物门 |
